# Exercise 01 — Terraform Basics

Fix a broken Terraform configuration that provisions an AWS EC2 instance.

## Quick Start

```bash
# Initialize providers
terraform init

# Validate syntax
terraform validate

# Preview changes
terraform plan

# Apply (requires AWS credentials)
terraform apply
```

## Files

- `starter/main.tf` — broken Terraform config (5 bugs)
- `solutions/main.tf` — fixed config
- `solutions/step-by-step-solution.md` — explanation of each fix
