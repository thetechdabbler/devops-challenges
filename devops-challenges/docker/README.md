## Prerequisites

- Docker Desktop installed and running
- Basic Linux terminal familiarity
- The sample app from `shared-resources/app/` (used in most exercises)

## Tools Covered

- Docker CLI (`build`, `run`, `exec`, `logs`, `inspect`)
- Dockerfile instructions
- Docker Compose
- Docker Hub / GitHub Container Registry (GHCR)
- `docker buildx` for multi-platform builds
