## Prerequisites

- Python installed (`ansible` is a Python package)
- `ansible` installed (`pip install ansible`)
- At least one target host (Docker container as a target works for most exercises)

## Tools Covered

- Ansible CLI (`ansible`, `ansible-playbook`, `ansible-vault`, `ansible-galaxy`)
- Inventory files and group variables
- Jinja2 templating
- Ansible Vault
- Ansible Galaxy
- Dynamic inventory scripts
